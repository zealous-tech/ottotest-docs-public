# Intro

Discover **Ottotest — an AI-powered, all-in-one testing platform** designed to simplify and accelerate your entire testing workflow.

## Getting started

Begin by **creating a new test case** in one of the following ways:

- Generate it from requirements using the AI assistant  
- Import it from a CSV file  
- Create it manually

Once the test case is reviewed, finalized, and ready to be automated, you can start the automation process.

## From Test Case to Automated Test

Ottotest makes it easy to convert **plain English test cases into automated tests**. Simply describe your testing scenario using clear steps and expected results, and the AI assistant can transform it into executable automation scripts.

This approach allows QA engineers, developers, and product teams to focus on test **logic and coverage**, while Ottotest handles the automation layer.

## Test Execution

Automated tests can be executed in different environments depending on your workflow:
- **Local execution** — run tests directly on your local machine during development and debugging.
- **Cloud execution** — execute tests in the Ottotest cloud environment for scalable, consistent test runs across projects.

This flexibility allows teams to test quickly during development and reliably during CI/CD pipelines.
