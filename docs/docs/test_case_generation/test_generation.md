# Intro

Ottotest provides multiple ways to create test cases, allowing teams to quickly document and organize their testing scenarios.

There are three ways to create a new test case in the Ottotest platform:
- **Generate from requirements** using the AI assistant
- **Import from a CSV** file
- **Create a test case manually**

All three methods allow you to generate test cases that document testing scenarios within the system. These test cases serve as a reference for test execution, help maintain structured QA documentation, and support the overall quality assurance process.

Once a test case is designed, reviewed, and ready for use, it can also be used as a foundation for test automation within Ottotest.
